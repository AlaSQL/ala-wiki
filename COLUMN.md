# Keyword `COLUMN`



Syntax:
```sql
    -- Result modifier for a `SELECT`  
    COLUMN OF SELECT ...
    SELECT COLUMN ...

    -- Normal SQL
    ALTER TABLE table ADD COLUMN column type
    SELECT * REMOVE COLUMN (column | LIKE mask)
```
