# Functions

1. Compile-time standard library
 * String functions
 * Number functions
 * Logical functions
 * Date-time functions
2. Run-time standard library
 * [ROWNUM()](ROWNUM) (or [ROW_NUMBER()](ROWNUM))
 * [COALESCE()](Coalesce)
3. User-defined functions
4. Into-functions
5. From-functions

Samples:
```sql
    ABS(), IIF(), IFNULL(), INSTR(), LOWER(), 
    UPPER(), LCASE(), UCASE(), LEN(), LENGTH()
    YEAR(date), DAYOFWEEK(date)
```
